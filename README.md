## Overview
This Power BI dashboard analyzes Netflix content based on:
- Content Type (Movies vs TV Shows)
- COVID Period (Pre-COVID vs During-COVID)
- Top Countries by Content
- Ratings Distribution
- Release Trends Over Time

## Tools Used
- Power BI
- DAX
- Netflix Dataset

## Key Insights
- Majority of Netflix content was released during COVID period
- Movies dominate over TV Shows
- United States has the highest content count
- Rapid growth after 2015
